# POS Üretim ve Fire Takip - Şifresiz Sürüm

Bu paket şifresiz sürümdür. Giriş ekranı yoktur.

## Dosyalar

- `app.py`: Uygulama kodu
- `requirements.txt`: Gerekli Python paketleri
- `ayar_dosyasi.xlsx`: Uygulamanın otomatik okuduğu Excel ayar dosyası
- `run_local.bat`: Windows'ta çift tıklayarak yerelde açmak için

## Yerelde çalıştırma

Klasörün içinde CMD/PowerShell açıp:

```bash
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

veya Windows'ta `run_local.bat` dosyasına çift tıklayın.

## Not

Şifresiz sürümde linki bilen herkes uygulamaya girebilir. Siteye yayınlarken bunu dikkate alın.
